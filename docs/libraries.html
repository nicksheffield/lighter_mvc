Libraries

app/libraries/

When you need more functionality than Lighter typically provides, it's possible to add libraries.

Lighter already comes with a selection of useful libraries that you can load and use, but it might not cover all your bases.

To install new libraries, you just need to add them into the app/libraries folder, and then they will be loadable.

Here is a list of existing libraries that come with Lighter.

  • Loader
  • URL
  • Input
  • Registry
  • Authentication
  • Database
  • Cart
  • Email
  • Form
  • Hash
