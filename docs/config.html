Config

app/config/config.php

The config file defines a set of options used by the application at various stages.

Below is a breakdown of each option.

default_controller
The default controller to be loaded when one is not specified in the url.
base_url
The url to the public folder. If the public folder is the web root (public_html) then this should be left blank.
autoload libs
An array of libraries to load automatically every time the website is loaded.
For example, to always load the form, cart, and email libraries, this setting would look like $config['autoload']['libs'] = array('form', 'cart', 'email');
autoload models
An array of models to load automatically every time the website is loaded. Syntax is exactly the same as autoloading libraries.
database credentials
The information needed to successfully connect to the database.
