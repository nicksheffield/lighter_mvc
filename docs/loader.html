Loader

sys/libraries/load.php

This library is used to load in views, libraries, models, and error pages whenever necessary.

Methods

model($name)
Static. Load a model from the app/models folder. Php file extension not needed.
library($name)
Static. Load a library from the app/libraries folder. Php file extension not needed.
view($name, $data)
Static. Load a view from the app/views folder. Extracts the $data array. Php file extension not needed.
error($name, $data)
Static. Load a view from the app/errors folder. Extracts the $data array. Php file extension not needed.
