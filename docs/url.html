URL

sys/libraries/url.php

This library is used to load in views, libraries, models, and error pages whenever necessary.

Methods

redirect($url)
Static. Redirect to $url.
parts()
Static. Returns an array of each segment of the url. eg, on home/login URL::parts() would return array('home', 'login').
part($n)
Static. Return a single part from the parts() method above.
string()
Static. Returns the current url in string form.
save($url = false)
Static. Stores the current url temporarily, or, if a url is provided, store that instead.
restore()
Static. Redirects to the url stored by save().
