Email

app/libraries/email.php

Create and send emails.

Properties

$to
String. The email address to send the email to.
Can be a single email address: bob@example.com
Can include a name: Bob <bob@example.com>
Can be a comma separated list of addresses: bob@example.com, john@example.com
Which can also include names: Bob <bob@example.com>, John <john@example.com>
$from
String. The from address
$subject
String. The subject of the email
$message
String. The main message
$html
Boolean. If this is set to true, all html inside the $message string will be treated as code
$success
Boolean. If the email is sent successfully, this is true

Methods

send()
Send the email, and return the success as a boolean
